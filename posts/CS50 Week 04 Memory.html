Hexadecimal

It’s base 16 which means there’s 16 digits you can use to represent a number.

0 to 9 plus A, B, C, D, E, F which equals 10, 11, 12, 13, 14 and 15.

You can represent two digits, each represent a column, 1 and 16, so for example A under the 1 column is 10 and 1 under the 16 column is 16.

FF is the highest value in this system: 16 * 15 + 15 * 1 = 255

The prefix 0x is used in code to indicate that the number is being written in hex.

& get you the address of a value in memory.

A pointer is a variable that points to the address of another variable.

A string variable is just a char pointer that refers to the address of the first char in a string and thanks to the null character we can follow the bread crumbs till we find the end of the string and get all the characters. 

char *s = "Hello, world!"; 

// [1] get the address of the first char 
// [2] go to memory and find the address 
// [3] access all values until the null character \0

Once you create a pointer you can get the value stored in the address you’re pointing at using *

char *name = "Hamza"; 
int *p = &name; // pointer 

printf("My name is %c\n", *p); // access value in the pointer address

You can do arithmetic with pointer, since a string is an array of character back to back, then the next character is plus 1 from the current character address. 

char *fruit = "apple"; 
char one = *fruit; 
char two = *(fruit + 1); 

printf("second chraceter in apple is : %c\n", two); // p

The computer memory gives you a segment you can use if you try to pass to the rest of memory you will get an error.

malloc() is a function in C that let us use a chunk of memory by passing the number of bytes we need as an argument.

In design it’s not good to call a function inside a loop because it’s stupid to call the function in each iteration when you could called it once and store the returned value inside a variable.

NULL pointer is a pointer that does not point to any memory location “an invalid memory location”.

If you pass NULL to a variable it become a NULL pointer.

NULL is useful for handling malloc() errors.

free() is a function we use to free the chunk of memory we have been using from malloc().

Any time you use malloc() you have to use free().

If you don’t initialize variables you will get garbage values, values from other programs that were running in that part of memory.

When you pass values to a function, you are only providing copies.

The heap is free available space in memory that we can use malloc() to use.

The stack is the variables and arguments passed to a function.
