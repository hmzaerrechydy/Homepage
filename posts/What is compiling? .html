What is compiling?


Here’s what I’ve learned this morning from CS50 Intro to CS Week 2 Arrays lecture:

  1. Preprocessing: adding external code from libraries you’ve used to your code.
  2. Compiling:
    1. Convert your language (c, c++, java, …) to assembly.
    2. Assembly is more low level, it helps writing more efficient code.
  3. Assembling: turn assembly code to binary.
  4. Linking: merging your code binary with other people’s code binary.
